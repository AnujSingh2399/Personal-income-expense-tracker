# Personal-income-expense-tracker
![GitHub repo size](https://img.shields.io/github/repo-size/AnujSingh2399/Personal-income-expense-tracker)

![Dashboard](income-expense-tracker.xlsx)

This is a basic Excel project I created to track monthly income, expenses, and savings.

🔹 Features:

* Monthly income tracking (Salary, Rental, etc.)
* Categorized expense tracking (Housing, Food, Transportation)
* Automatic total calculations
* Savings/Deficit calculation
* Year-to-date summary

🔹 Tools Used:

* Microsoft Excel
* Basic formulas (SUM, formatting)

🔹 Purpose:
I built this project to improve my Excel skills and understand how financial tracking systems work.

🔹 Future Improvements:

* Adding charts and dashboards
* Automating calculations further
* Improving UI/UX

An Excel-based tracker to manage monthly income, expenses, and savings with categorized analysis.

## ⚠️ Note
This project uses simulated data for learning purposes.
